# VarSync

Sync variables. Preserve connections.

## 🚀 Overview

VarSync is an open-source Figma plugin designed to make variable workflows reliable.

It allows you to copy and manage variable collections across files while preserving alias relationships and structure.

## ❗ Problem

Existing tools:

* Break variable connections
* Fail on large collections
* Lack reliability and transparency

## ✅ Solution

VarSync ensures:

* Safe variable copy and paste
* Alias relationships are preserved
* Stable performance across files

## 🔥 Features (MVP)

* Copy variable collections
* Paste variable collections
* Preserve alias connections

## 🛠 Installation

1. Open Figma Desktop
2. Plugins → Development → Import plugin
3. Select manifest.json
